# FlashSales
* It is created with ruby on rails 5.2.3

## Technolgy used:

* ruby version - 2.6.3
* rails version - 5.2.3'
* download ruby on rails and any query follow https://rubyonrails.org/
* postgresql database

## do the following:

* rails db:create
* rails db:migrate
* rails db:seed
* rails s //for running server
* now open in browser http://localhost:3000
